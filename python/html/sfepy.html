1. Installation prerequisites

1.1 Installation prerequisites

  • a C compiler suite(for example gcc)
  • python2.7 or 3
  • NumPy
  • Cython

1.2 Python packages required

  • Pyparsing
  • SciPy
  • scikit-umfpack
  • Matplotlib
  • PyTables
  • sympy
  • mayavi
  • pysparse
  • igakit
  • petsc4py
  • mpi4py
  • pymetis

1.2.1 Pyparsing

安装：

pip install pyparsing

1.2.2 SciPy

安装：

pip install scipy

1.2.3 scikit-umfpack

Dependencies

scikit-umfpack depends on NumPy, SciPy, SuiteSparse, and swig is a build-time dependency.

Building SuiteSparse
sudo apt-get install python3-dev
sudo apt-get install libsuitesparse-dev
Get source code and install
git clone https://github.com/scikit-umfpack/scikit-umfpack.git
cd scikit-umfpack
python setup.py install

pip install matplotlib

若提示没有安装vtk，则
[1]download and install cmake 

sudo apt-get install cmake
[2]download vtk source and compile and install 
wget http://www.vtk.org/files/release/7.1/VTK-7.1.1.tar.gz
tar xvzf VTK-7.1.1.tar.gz 
cd VTK-7.1.1
mkdir vtk-build 
cd vtk-build 
cmake ..
make
make install

[3]download and install vtk python interface 

wget http://www.vtk.org/files/release/7.1/vtkpython-7.1.1-Linux-64bit.tar.gz
tar xvzf vtkpython-7.1.1-Linux-64bit.tar.gz 
cd vtkpython-7.1.1-Linux-64bit
