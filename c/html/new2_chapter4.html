4.2 关系运算

4.2.1 关系运算符及其优先次序

运算符 意义 优先级
< 小于 高
<= 小于等于 高
> 大于 高
>= 大于等于 高
== 等于 低
!= 不等于 低

关系运算的结果是假(false，C里面用0表示)，或真(true，C里面用 1表示)。

关系运算的优先级是低于算术运算，高于赋值运算


            int a , b, c;
            a = b > c + 1; //先算c+1, 然后是 >, 然后是 =

相同等级的关系运算符，结合律是从左到右。

注意：不能使用连续的关系运算表示区间：


          int a = -3;
          if ( -6 < a < -2 )...;
